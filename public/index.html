Project 2

Developed a face recognition model that leverages unlabeled data to boost accuracy. By employing PCA for dimensionality reduction and using KNN, ANN, and LR algorithms, the project achieved an 83.16% accuracy with KNN by iteratively adding unlabeled samples. This approach demonstrates a practical solution for real-world applications where labeled data is limited.
