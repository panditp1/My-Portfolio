My Projects

Project 1

I built a pet shelter management app to make life easier for shelters. The app has a simple interface to track pets and adoptions, using HTML, CSS, and jQuery for the front end. On the back end, I used Node.js and MongoDB to keep everything running smoothly and handle data efficiently. This project was a great learning experience, especially in designing user-friendly tools and managing data effectively.

Project 2

I worked on a face recognition model that uses unlabeled data to improve accuracy. By using PCA for reducing dimensions and algorithms like KNN, ANN, and Logistic Regression, I got the best result with KNN at 83.16% accuracy by iteratively adding unlabeled samples. This project taught me a lot about working with limited labeled data and solving real-world challenges with creative solutions.

Project 3

I created a Pantry App Tracker to make keeping track of pantry items super easy. It helps users know what they have, sends reminders before items expire, and even suggests recipes based on what’s in stock. I built it using React.js for a smooth, user-friendly interface and Firebase to keep everything synced and accessible. It’s all about saving time, money, and reducing food waste.

Project 4

As someone who’s passionate about the universe and exploring concepts like time and space, I’m building a full-stack web app called Cosmic Explorer. The app lets users dive into ideas like time dilation and cosmic distances through interactive tools. I’m using React.js on the frontend to make it user-friendly and Node.js with Express.js on the backend for handling calculations and APIs. Tools like Nodemon help streamline the process by auto-restarting the server.
